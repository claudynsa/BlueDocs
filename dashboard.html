Accessibility

Sesuaikan tampilan dan cara Anda menjelajahi BlueDocs.

Pembaca Layar

Saat aktif, elemen yang difokuskan (Tab) atau diklik akan dibacakan.

High Contrast Mode

Tingkatkan kontras warna untuk keterbacaan lebih baik.

Font Size

Perbesar atau perkecil ukuran huruf di seluruh halaman.

Keyboard Navigation

Seluruh menu dan tombol di BlueDocs dapat diakses tanpa mouse.

Gunakan Tab / Shift+Tab untuk berpindah, Enter atau Space untuk memilih, dan Esc untuk menutup panel ini.
